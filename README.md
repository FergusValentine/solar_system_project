# Solar System Simulation

A 2D solar system simulation built in Python using Pygame.

The project simulates planetary motion using Newtonian gravity, with planets interacting through gravitational attraction. The simulation includes camera controls, zooming, panning, and a MVC-based architecture.

## Features

- Newtonian gravity simulation
- Multiple planetary bodies
- Orbital movement
- Camera system with:
  - Zoom
  - Panning
